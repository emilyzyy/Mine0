# Minecraft Screenshot Drop Folder

Put sample first-person Minecraft screenshots in this folder.

Recommended naming:

- `step_001.png`
- `step_002.png`
- `step_003.png`

Supported formats:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`

Selection behavior:

- Files are sorted lexicographically with numeric ordering.
- The first run step uses the first file.
- If there are fewer screenshots than requested steps, the last screenshot is reused.

Recommended screenshot contents:

- Full Minecraft game window or fullscreen capture
- First-person view only
- Keep the hotbar visible
- Keep hearts / hunger visible if possible
- Avoid capturing your entire desktop if you can
- Avoid overlays that cover the crosshair or terrain

If you only have one screenshot, use `step_001.png` and the app will reuse it.
